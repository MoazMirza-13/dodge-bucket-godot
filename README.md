# Dodge Bucket
A simple Godot 4 project template written in GDScript where the player dodges falling rocks. Great for learning Godot basics, testing mechanics, or using as a starter template.

## Overview
In **Dodge Bucket**, rocks fall from above while the player must move the bucket left or right to avoid getting hit. The challenge increases as the screen scrolls upward, testing your reflexes and timing.

## Features
- Player movement and controls (touch screen based)  
- Falling rock spawning system  
- Smooth upward screen motion  
- Basic physics and collision logic  

## Built With
- **Godot Engine** (v4.x recommended)  
- **GDScript**

## Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/MoazMirza-13/dodge-bucket-godot.git
   ```
2. Open the project in Godot Engine.
3. Press Play ▶️ to start
